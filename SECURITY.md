# Security Policy

SenseSight is early-stage open source software. Please do not file public issues
for vulnerabilities that could expose users, credentials, infrastructure, or
robot data.

## Reporting

Report sensitive security concerns privately to the maintainers through GitHub.
Include:

- affected package or app
- reproduction steps
- expected impact
- any known mitigations

## Supported Versions

The `main` branch is the only supported development line until the project tags
its first release.
